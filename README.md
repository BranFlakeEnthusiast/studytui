# studytui
collection of tools created to be helpful for my productivity and studying

tab and shift+tab can be used to navigate between modules

## todo list

by default the todo list is stored in ~/Documents/notes/tasks.json
this can be chaned in main.go where todo.New is called

keybinds:

up: ↑/k\
down: ↓/j

quit: q/ctrl+c\
add: a\
delete: d/backspace\
toggle: space/enter/x\
edit: e

## pomodoro timer

this uses a 25 minute work period followed by a 5 minute break period

quit: q/ctrl+c\
play/pause: space\
reset: r\
switch mode: s
